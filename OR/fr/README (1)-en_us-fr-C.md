# HTML2DOC

Une solution rapide pour convertir les documents HTML en DOC pour le traitement des fichiers Cision. Ajout d'un texte pour la traduction.

## Pour commencer

Ces instructions vous permettront d'obtenir une copie du projet sur votre machine locale à des fins de développement et de test. Voir déploiement pour savoir comment déployer le projet sur un système actif.

### Conditions préalables

Les éléments nécessaires à l'installation du logiciel.

* Cadre .NET

### Installation

Un guide pas à pas qui vous explique comment mettre en place l'environnement de développement et le faire fonctionner.  Aucune installation n'est nécessaire. Il suffit d'exécuter le fichier .exe ```
