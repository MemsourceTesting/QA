# HTML2DOC

Eine schnelle Lösung zur Konvertierung von HTML-Dokumenten in DOC für die Bearbeitung von Cision-Dateien. Hinzufügen von Text für die Übersetzung.

## Erste Schritte

Mit diesen Anweisungen erhalten Sie eine Kopie des Projekts, die auf Ihrem lokalen Rechner zu Entwicklungs- und Testzwecken läuft. Siehe Bereitstellung für Hinweise zur Bereitstellung des Projekts auf einem Live-System.

### Voraussetzungen

Die Dinge, die Sie vor der Installation der Software benötigen.

* .NET Framework

### Einrichtung

Eine Schritt-für-Schritt-Anleitung, die Ihnen zeigt, wie Sie die Entwicklungsumgebung zum Laufen bringen.  $ Keine Installation erforderlich. Führen Sie einfach die .exe ```
