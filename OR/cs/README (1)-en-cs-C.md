# HTML2DOC

ZMĚNA!!! - Rychlé řešení pro převod dokumentů HTML do DOC pro zpracování souborů Cision. Přidání textu pro překlad.

## Začínáme

Tyto pokyny vám poskytnou kopii projektu spuštěného na místním počítači pro účely vývoje a testování. Poznámky k nasazení projektu v živém systému najdete v části Nasazení.

### Předpoklady

Věci, které potřebujete před instalací softwaru.

* .NET Framework

### Instalace

Průvodce krok za krokem, který vám řekne, jak zprovoznit vývojové prostředí. ```$ Není nutná žádná instalace. Jednoduše spusťte soubor.exe ```
